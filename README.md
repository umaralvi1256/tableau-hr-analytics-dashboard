# HR Workforce Analytics Dashboard

## Project Overview
This project involves the development of an interactive Tableau dashboard designed to visualize key Human Resource metrics. The primary objective was to transform raw organizational data into actionable insights to assist management in tracking workforce trends, identifying turnover patterns, and assessing departmental performance.

## Key Features
* **Workforce Overview:** A high-level summary of headcount, departmental distribution, and demographic data.
* **Turnover Analysis:** Visual representation of employee attrition rates to help identify areas of concern.
* **Performance Tracking:** An interactive interface to filter performance metrics across different roles and tenures.

## Business Value
The dashboard enables HR stakeholders to move beyond manual reporting, allowing for faster, data-driven decision-making regarding recruitment strategies, talent retention, and organizational health.

## Tools Used
* **Tableau:** Used for data visualization and building interactive dashboards.
* **Microsoft Excel:** Used for data cleaning and preparation.
